\section{Discussion}
Results show that PCR-induced chimeras in short-read mitochondrial data can be detected and effectively removed using interpretable features derived from standard alignments, and that a sequence-based CNN can further improve sensitivity. The Gradient Boosting model is conservative, achieving high precision for the chimeric class but missing more chimeric reads. The CNN1D substantially improves recall and ROC-AUC, but also increases the number of clean reads incorrectly flagged as chimeric, which can reduce read coverage when the read depth is low. 
Downstream assembly evaluation indicates that optimal filtering depends on both coverage and chimera percentage. At moderate-to-high coverage and high chimera rates, highly sensitive chimeric filtering can simplify graphs and restore near-contiguity. At minimal depth, however, aggressive filtering may remove critical read support and introduce fragmentation. These findings suggest that deployment should show a tunable threshold and report read-retention diagnostics to support user calibration.

\section{Discussion}
Results show that PCR-induced chimeras in short-read mitochondrial data can be detected using lightweight, interpretable features derived from standard alignments, and that a sequence-based CNN can further improve sensitivity. The Gradient Boosting model is conservative, achieving high precision for the chimeric class but missing more chimeric reads. The CNN1D substantially improves recall and ROC--AUC, but increases false positives, which can reduce read support when coverage is low.

Downstream assembly evaluation indicates that optimal filtering depends on both coverage and chimera burden. At moderate-to-high coverage and high chimera rates, stronger filtering can simplify graphs and restore near-contiguity. At minimal depth, however, aggressive filtering may remove critical read support and introduce fragmentation. These findings suggest that deployment should expose a tunable threshold and report read-retention diagnostics to support user calibration.

Limitations include reliance on simulation and a single target species, and the use of a single alignment strategy for feature extraction. Broader validation across taxa, library preparation conditions, and empirical datasets is required to establish generalizability. Future work includes calibration on real sequencing libraries, evaluation under varying sequencing error profiles, and extension of the CLI workflow to support additional model families and references.

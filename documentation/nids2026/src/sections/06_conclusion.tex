\section{Conclusion}
MitoChime provides a practical pre-assembly chimera detection pipeline for mitochondrial Illumina paired-end reads using interpretable alignment- and sequence-derived signals. A tuned Gradient Boosting classifier achieves robust performance with conservative filtering, while a CNN1D model achieves higher sensitivity and discrimination. External assembly evaluation demonstrates that filtering can substantially reduce assembly graph complexity under high chimera burdens, with coverage-dependent trade-offs at low depth. These results support MitoChime as a quality-control step that can improve mitochondrial reconstruction workflows in resource-constrained settings.

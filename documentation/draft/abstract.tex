\begin{center}
\textbf{Abstract}
\end{center}

\setlength{\parindent}{0pt}
Next-generation sequencing (NGS) has advanced genomic research but remains vulnerable to PCR-induced chimeras that can compromise mitochondrial genome assembly. These artifacts are particularly problematic for small, circular mitochondrial genomes, where they can produce fragmented contigs and false junctions. Existing detection tools are primarily designed for amplicon-based microbial data and rely on reference databases or abundance assumptions unsuitable for organellar assembly.

This study presents MitoChime, a machine learning pipeline for detecting PCR-induced chimeric reads in \textit{Sardinella lemuru} Illumina paired-end data. Simulated clean and chimeric reads were generated, aligned, and used to build labelled datasets for both feature-based and sequence-based models. Classical machine learning methods were benchmarked alongside a one-dimensional convolutional neural network (1D CNN) and a Bidirectional Gated Recurrent Unit (BiGRU).

For classical machine learning, the Tuned Gradient Boosting (GB) performed best (F1 = 0.7765, ROC-AUC = 0.8459), with clipping-based and k-mer divergence features contributing most. However, overal deep learning models performed better, with 1D CNN (F1 = 0.9135, ROC-AUC = 0.9607) and BiGRU achieving the highest performance (F1 = 0.9523, ROC-AUC = 0.9849). Downstream assembly tests showed improved structure after filtering, though the top model did not always yield optimal assembly continuity. These results highlight MitoChime as an effective pre-assembly filtering approach.

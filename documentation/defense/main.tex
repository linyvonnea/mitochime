\begin{block}{Assembly quality}
\begin{itemize}
    \item Fragmented or failed mitochondrial assemblies
    \item Reduced reliability of mitochondrial genomics workflows
\end{itemize}
\end{block}

\vspace{0.8em}

\begin{block}{Research impact}
\begin{itemize}
    \item Limited downstream analyses in fisheries, population genetics, and evolution
    \item Wasted sequencing effort, computing time, and researcher labor
\end{itemize}
\end{block}

% Slide 5: Existing approaches
% -------------------------------------------------
\begin{frame}{Existing approaches}

\scriptsize
\renewcommand{\arraystretch}{1.35}

\begin{table}
\centering
\begin{tabular}{p{0.23\textwidth} p{0.30\textwidth} p{0.37\textwidth}}
\hline
\textbf{Tool / Approach} & \textbf{Main focus} & \textbf{Gap for this study} \\
\hline
UCHIME / UCHIME2 &
Amplicon chimeras &
Reference/abundance dependent \\

DADA2 &
ASV-level filtering &
Not read-level mtDNA filtering \\

CATCh &
16S rRNA ensemble ML &
Not for mtDNA; inconsistent \\

ChimPipe &
RNA-seq fusion/chimera detection &
Not PCR-induced mtDNA artifacts \\

DL-based tools &
DNA sequence motif prediction &
Few to no tools for chimera detection \\

\textbf{MitoChime} &
\textbf{Read-level mtDNA chimeras} &
\textbf{Pre-assembly ML/DL filtering} \\
\hline
\end{tabular}
\end{table}

\vspace{0.5em}

\centering
\footnotesize
Most existing tools target amplicon, ASV, or RNA-seq chimeras; MitoChime targets PCR-induced mitochondrial short-read artifacts.

\end{frame}

% -------------------------------------------------
% Slide 7: Study problem
% -------------------------------------------------
\begin{frame}{Study problem}

\begin{center}
\Large
Can machine learning detect PCR-induced chimeric reads\\
\keyword{before mitochondrial genome assembly}?
\end{center}

\vspace{1.5em}

\begin{columns}[T]
\column{0.48\textwidth}
\begin{block}{Input}
Simulated clean and chimeric paired-end Illumina reads from \textit{S. lemuru}
\end{block}

\column{0.48\textwidth}
\begin{block}{Output}
A modular pipeline that predicts and filters suspected chimeric reads
\end{block}
\end{columns}

\end{frame}

% Slide 8: Scope and limitations

% -------------------------------------------------
% Slide 22: Recommendations
% -------------------------------------------------
\begin{frame}{Recommendations: Practical Use}

\begin{itemize}
    \item Use \textbf{Gradient Boosting} when read depth is low.
    \item Use \textbf{BiGRU} when read-level classification is the priority.
    \item \textbf{CNN} may be useful in high-coverage and high-chimera conditions.
    
\end{itemize}

\end{frame}

\begin{frame}{Recommendations: Future Work}

\begin{itemize}
    \item Test on real mitochondrial datasets.
    \item Evaluate realistic class imbalance.
    \item Add varied sequencing-error profiles.
    \item Explore hybrid feature-sequence models.
\end{itemize}

\end{frame}

% -------------------------------------------------
% Slide 23: Summary
% -------------------------------------------------
\begin{frame}{Summary}

\large
\begin{enumerate}
    \item PCR-induced chimeras can disrupt mitochondrial genome assembly.
    \item Existing tools do not directly address this read-level setting.
    \item MitoChime detects suspected chimeric reads before assembly.
    \item BiGRU achieved the strongest classification performance.
    \item Model choice should consider both accuracy and read retention.
\end{enumerate}

\end{frame}

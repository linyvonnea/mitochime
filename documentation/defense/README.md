# MitoChime Beamer Defense Deck

## Compile

From this folder:

```bash
latexmk -pdf main.tex
```

Alternative:

```bash
pdflatex main.tex
pdflatex main.tex
```

## Notes

- All figures needed by `main.tex` are already included in `figures/`.
- Replace any placeholder figure with your real manuscript figure using the same filename.
- The final output will be `main.pdf`.
